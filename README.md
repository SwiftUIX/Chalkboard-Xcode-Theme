### Installation

1. Quite Xcode
2. Move theme file to: `~/Library/Developer/Xcode/UserData/FontAndColorThemes/` (if the last folder doesn't exist, create it).
3. Open Xcode > `Preferences` > `Fonts & Colors` and select `Chalkboard`.
